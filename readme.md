# choco-ui

## Install

`npm install -S @qiqzhao/choco-ui`

## Usage

npm

```
import { Button } from "@qiqzhao/choco-ui";
import "@qiqzhao/choco-ui/dist/style.css";

<Button type="primary">
  <span>click</span>
</Button>
```
